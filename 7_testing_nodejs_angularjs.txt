Karma.js
    - Ejecuta los test de JS según se van construyendo, 
    de forma que ante cualquier fallo el desarrollador se dará cuenta de inmediato.
    - Se configura con Jasmine.js como framework de testing. 

Jasmine
    - Es un framework Behavior Driven Development para realizar Unit Testing. 
    - La herramienta de testeo en Angular por defecto es Jasmine.
    - https://desarrolloweb.com/articulos/conociendo-jasmine.html

Protractor
    - end-to-end testing framework for AngularJS.
    - Verifica que los componentes de una aplicación funcionan correctamente entre sí.
